# Counting in Binary and Hex

Binary aka base 2

The power of 2's

| 0   | 0   | 0   | 0   | 0   | 0   | 0   | 0   |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 128 | 64  | 32  | 16  | 8   | 4   | 2   | 1   |

Hexa aka base 16

## Opdrachten

Decimaal -> binair
| Decimal | Binary |
| --------| ------- |
| 16 | 0001 0000 |
| 128 | 1000 0000|
| 228 | 1110 0100|
| 112 | 0110 1100|
| 73 | 0100 1001|

Binair -> decimaal
| Binair | decimaal |
| ----- | ------ |
| 1010 1010 | 170 |
| 1111 0000 | 240 |
| 1101 1011 | 218|
| 1010 0000 |160 |
| 0011 1010 | 50|

Decimaal -> Hex
| Decimaal | Hex |
| ----- | ------- |
| 15 | 15/16 % 15 -> 15 -> F |
| 37 | 37/16 % .5, 2/16 % 2 -> 5 & 2 -> 25 |
| 246 | 246/16 % .6, 15/16 % 15 -> 6 & 15=f -> F6|
| 125 |125/16 % .13, 7/16 % 7 -> 13=d & 7 -> 7d|
| 209 |209/16 % .1, 13/16 % 13 -> 1 & 13=d -> d1 |

Hex -> Decimaal
| Hex | Decimaal |
| -----| ------ |
| 88 | 8*16 + 8*16 = 136 |
| e0 | e=14*16 + 0*16 = 224|
| cb | c=12*16 + 11*16 = 203|
| 2f | 2*16 + 15*16 = 47|
| d8 | d=13*16 + 8*16 = 216|
